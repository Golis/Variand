Variand

Variand, an Android app to make an analysis about heart rate variability

Download this project as a .zip file Download this project as a tar.gz file

Description

Variand is an Android app that will let you make a specific graphic analysis about heart rate variability. This app is intended to install and execute in an Android tablet and it was developed and designed for researchers or clinical professionals without requiring special skills in computer science.

In addition, Variand allows to:

  • Load data from a .txt
  • Show different graphics using this data
  • Manipulate these graphics zooming and scrolling
  • Apply different kind of filters over the signal
  • Allows to perform both spectral and nonlinear analysis of several parameters related to the cardiac rhythm
  • Export the graphic to a .png or a .jpeg
  • Sync with dropbox to upload and download documents and data
  • Allows the management of the sdcard from the app
