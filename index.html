Description

Variand is an Android app that will let you make a specific graphic analysis about heart rate variability. This app is intended to be installed and executed in an Android tablet and it was developed and designed for researchers or clinical professionals and it does not require special computer science skills.

In addition, Variand allows you to:

  • Load data from a .txt
  • Show different graphics using these data
  • Manipulate these graphics zooming and scrolling
  • Apply different kinds of filters to a specific signal
  • Perform both spectral and nonlinear analysis of several parameters related to the cardiac rhythm
  • Export the graphic to a .png or a .jpeg
  • Sync with Dropbox to upload and download documents and data
  • Manage the sdcard from the app

Demo of Variand running in an Android tablet

Requirements

Hardware requirements: Variand users only need an Android tablet. It is important to know that the more memory the processor tablet has the more the performance of the application will be

Software requirements: Variand is compatible with most versions of Android. In fact, this app is compatible with versions after 2.2

Installing Variand

If you want to try Variand you should download the code in tar.gz or .zip format and make an .apk with the Android SDK
